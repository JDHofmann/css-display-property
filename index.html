CSS Display Property

Formally, the display property sets an element's inner and outer display types. The outer type sets an element's participation in flow layout; the inner type sets the layout of children.

Display: none

Display: none means the element is completely removed. Display: inline means an element displays inline. Height and width properties will have no effect

DISPLAYING

Display: block means the element starts on a new line, and takes up the whole width. Display: inline-block will display an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values.

Display: inline

Display: none means the element is completely removed. Display: inline means an element displays inline. Height and width properties will have no effect
